# Node.js 에서 파일목록

## 문제

application은 여전히 몇 가지 문제점이 있다.

데이터를 추가하려면 여기 있는 글 목록을 직접 수정해야 한다.

리스트의 추가, 삭제가 빈번하다면 이를 어떻게 기계에 맡길지를 생각해보아야 한다.

## 해결 방안 모색

그러기 위해 먼저 Node.js 에서 파일 목록을 알아낼 수 있는지 확인해보자.

- 데이터 디렉토리에 파일이 추가 / 삭제 될 때 그것을 어떻게 node.js를 통해서 알아낼 수 있는가?
  -> nodejs file list in directory 검색

- readdir.js 를 실행시켜보면 `[ 'CSS', 'HTML', 'JavaScript' ]` 가 출력되는 것을 확인할 수 있다.
