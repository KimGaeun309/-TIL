-> nodejs file read 검색해보거나 node.js 공식 홈페이지에 들어가 모듈 중 File System 을 클릭해 확인

## fileread.js

- 이를 실행하려면 `$ cd ch12_file_read` 로 디렉토리를 이동한 다음 `$ node fileread.js` 를 해야 한다.
