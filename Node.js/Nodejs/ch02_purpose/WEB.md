# Node.js 를 사용하는 목적

## 1. html 파일을 쉽게 수정

- 예를 들어 이 웹페이지에서 순서 있는 목록을 순서 없는 목록으로 바꾸고 싶다면?
- 파일 하나하나 일일히 수정해야 할 경우 파일이 1억개면 힘들어진다
- 이런 상황에서 node.js를 활용하면 편하게 바꿀 수 있다.
- 즉, node.js 와 같은 것을 사용하면 하나의 파일에 있는 html 코드만 바꾸어도 여러 파일에 있는 코드를 한번에 바꿀 수 있는 효과를 가지게 된다.
- 이것이 가능한 이유는, 우리가 웹페이지 1억개를 저장한 것이 아니라 사용자가 어떠한 페이지를 요청할 때마다 node.js 와 같은 기술로 순간순간에 웹페이지를 프로그래밍적으로 생성해내기 때문에 가능하다.

## 2. 사용자가 컨텐츠를 쓰고 수정하고 삭제할 수 있다

- 또, node.js 와 같은 기술을 사용하기 전에는 사용자로부터 컨텐츠를 받을 수 없어 이메일을 통해 받아서 직접 html 파일을 만들어야 했었지만, node.js로 만든 웹페이지를 사용한다면 사용자로부터 받은 정보를 바로 추가할 수 있다.
- 우리는 node.js를 이용해서 우리 웹사이트로 방문하는 사용자가 컨텐츠를 읽기 뿐만 아니라 쓰기, 수정, 삭제 작업 모두 웹을 통해 할 수 있도록 제공할 수 있다.
